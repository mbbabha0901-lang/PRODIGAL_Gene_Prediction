# PRODIGAL_Gene_Prediction

## Overview
This repository contains gene prediction and annotation results obtained from the SPAdes-assembled *Lactiplantibacillus plantarum* genome using Prodigal.

## Project Information
- **Organism:** *Lactiplantibacillus plantarum* WCFS1
- **Assembler:** SPAdes
- **Gene Prediction Tool:** Prodigal v2.6.3
- **Genetic Code Table:** 11 (Bacterial and Archaeal)

## Input File
- `contigs.fasta` – Genome assembly generated by SPAdes

## Prodigal Command

```bash
prodigal -i contigs.fasta \
-a translated_sequences.faa \
-d nucleotide_sequences.fna \
-o annotations.gbk \
-g 11
```

## Output Files

| File | Description |
|------|-------------|
| annotations.gbk | Gene annotations in GenBank format |
| nucleotide_sequences.fna | Predicted coding nucleotide sequences |
| translated_sequences.faa | Predicted protein sequences |
| contigs.fasta | SPAdes-assembled contigs |
